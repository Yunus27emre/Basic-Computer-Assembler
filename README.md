# Basic-Computer-Assembler
Basic Computer Architecture – Assembler Implementation (Java)
Bu proje, temel bilgisayar mimarisindeki 25 komutu destekleyen basit bir Assembler uygulamasıdır.
Uygulama Program.txt dosyasını okuyarak sembolik komutları makine koduna çevirir ve hem hexadecimal hem de binary formatta çıktılar üretir.
